# Auto Generate Dalton Articles

WINDOWS ONLY! Generate Dalton Bioanalytics scientific summary articles with Claude Code. The app can run once immediately or run immediately and then continue on a recurring day interval.

## Quick Setup

1. Install Claude Code CLI.

2. Confirm Claude works from PowerShell:

```powershell
claude --version
```

3. Download and unzip:

```text
AutoGenerateDaltonArticles-windows.zip
```

from the latest version of
https://github.com/Xinye-Star-Yu/Auto_generate_dalton_articles/releases

5. Put the unzipped app somewhere where claude can recognize

6. Double-click:

```text
AutoGenerateDaltonArticles.exe
```

## Using the App

The GUI has two settings:

```text
Articles [N]
Repeat every [X] days
```

Then click `Start`.

- `Articles = 2`, `Repeat every = 0`: run 2 articles immediately and do not create a schedule.
- `Articles = 2`, `Repeat every = 7`: run 2 articles immediately, then run 2 more every 7 days.
- `Articles = 1`, `Repeat every = 1`: run 1 article immediately, then run 1 article every day.

Logs appear in the GUI while it runs. The `DOI History` tab shows the DOI duplication CSV used to avoid reusing articles. You can add DOI rows manually or delete selected rows from that tab.

Outputs are created next to AutoGenerateDaltonArticles.exe. The only relevant output are the dated folders containing the articles.
